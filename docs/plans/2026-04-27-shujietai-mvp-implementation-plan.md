# ShuJieTai MVP Implementation Plan

> For Hermes: Use subagent-driven-development skill to implement this plan task-by-task.

Goal: Deliver a runnable MVP that unifies multi-platform conversation events and presents a three-column cockpit dashboard.

Architecture: Build a FastAPI backend with canonical event/session APIs and a Vue 3 frontend cockpit UI. Use Docker Compose as the single local orchestration point with PostgreSQL and Redis attached for future persistence and realtime extensions.

Tech Stack: FastAPI, Uvicorn, Pydantic, Pytest, Vue 3, Vite, Docker Compose, PostgreSQL, Redis.
